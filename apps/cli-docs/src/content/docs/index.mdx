## What is AgentFlow?

AgentFlow is a platform for building AI agent pipelines defined declaratively in YAML. A pipeline triggers on an event (e.g. a Stripe payment), orchestrates a DAG of `AgentPod` workers via LangGraph, and produces an output (deployed website, report, etc.).

The `agentflow` CLI gives you kubectl-style access to your pipelines, runs, and resources — directly from the terminal.

If you are developing **inside the AgentFlow monorepo**, use **`pnpm run af -- …`** from the repo root after building `@agentflow/sdk` — see [Installation](/getting-started/installation/).
